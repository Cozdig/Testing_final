Git

  • git init - создаёт новый локальный репозиторий в текущей папке
  • git add - добавляет изменения в индекс для последующего коммита
  • git commit -m "сообщение" - сохраняет изменения в репозитории с комментарием
  • git status - показывает состояние файлов (изменённые, добавленные и т.д.)
  • git log - выводит историю коммитов
  • git push - отправляет изменения в удалённый репозиторий
  • git pull - загружает изменения из удалённого репозитория
  • git clone - клонирует удалённый репозиторий на локальный компьютер
  • git branch - показывает список веток или создаёт новую
  • git merge - сливает изменения из одной ветки в другую
